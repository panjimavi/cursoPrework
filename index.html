Paso dos: Instalar WSL - Ubuntu

  • Abrir una ventana de powershell en modo administrador
  • Instala WSL con el siguiente comando // wsl --install //
  • Al finalizar la instalación pedirá reiniciar sistema
  • Una vez reiniciado Ubuntu se ejecuta automaticamente y se instala
  • Una vez instalado pedirá que ingreses usuario y despues contraseña (serán los datos de admin)

Para abrir el explorador de archivos usa el comando // explorer.exe . //

Para trabajar con derechos de administrador usar el comando // sudo //

Comando para dar permisos de acceso a una carpeta // sudo chown -R nombreUsuario ~/nombreCarpeta // ejemplo; sudo chown -R panjimavi ~/cursos //

Algunos comandos utlizados:

  • pwd // Para ver en que carpeta estamos
  • cd // Para navegar através de las carpetas
  • cd ~ // Para ir a la carpeta principal (panjimavi)
  • ls // Muestra los archivos que tenemos en la carpeta
  • ls -a // muestras los archivos ocultos
  • clear // Limpia la consola
  • cd .. // Para ir a la carpeta anterior
  • mkdir // Para crear una carpeta
  • rmdir // Para eliminar carpetas vacías
  • touch // para crear un archivo
  • rm -rf // para borrar una carpeta y los archivos que contenga
  • cp // Se usa para copiar un archivo o carpeta
  • mv // Se usa para mover un archivo o carpeta

Cuando termines de programar... No olvides apagar la terminal de ubuntu; Abra una ventana de comando (No linux) y escriba el siguiente comando // wsl --shutdown //

Paso tres: Instalar Node.js

Para instalar la última version dde nodejs y npm, ejecute en la terminal de ubuntu los siguientes comandos

  1. sudo apt-get update
  2. sudo apt-get upgrade
  3. sudo apt-get install -y nodejs
  4. sudo apt-get install npm
  5. sudo apt-get update
  6. sudo npm install -g n
  7. sudo n latest
  8. npm -v
  9. node -v

Paso cuatro: Instalar Git

Para instalar git, ejecute en la terminal de ubuntu los siguientes comandos

  1. git --version
  2. sudo apt-get update
  3. sudo apt-get upgrade
  4. sudo apt-get install git
  5. sudo add-apt-repository ppa:git-core/ppa
  6. git config --global user.name "panjimavi"
  7. git config --global user.email "fjja86@gmail.com"
  8. sudo apt-get update
  9. sudo apt-get install git
  10. git --version

Paso cinco: Vincular una llave SSH en Github

Para vincular una llave SSH a Github, ejecute en la terminal de ubuntu los siguientes comandos

  1. ssh-keygen -t rsa -b 4096 -C "fjja86@gmail.com"
  2. eval "$(ssh-agent -s)"
  3. cd ~/.ssh
  4. cat id_rsa.pub

Copia el codigo que arroja y pegalo en la pagina de github, en el apartado SSH and GPG keys

Paso seis: subir un repositorio

Primero debes crear un repositorio en Github con el mismo nombre de la carpeta donde estan tus archivos

Luego ejecuta en la terminal de ubuntu los siguientes comandos

  1. echo "# cursoPrework" >> README.md
  2. git init
  3. git add .
  4. git commit -m "first commit"
  5. git branch -M main
  6. git remote add origin git@github.com:panjimavi/cursoPrework.git
  7. git push -u origin main

Realizar prueba: edita cualquier cosa en VS, guardalo y realiza la actualización ejecutando en la terminal de ubuntu los siguientes comandos

  1. git add .
  2. git commit -m "prueba de cambios"
  3. git push -u origin main
